# 心理學知識庫

> **適用對象**：心理諮商師、臨床心理師、心理衡鑑專業人員；**AI Agent** 撰寫個案概念化報告之知識來源。  
> **用途**：臨床晤談、心理衡鑑、個案概念化、治療規劃。  
> **使用原則**：依實證、文化脈絡與個案特性整合；**僅根據提供之個案資料撰寫**，不得臆造；涉及診斷、用藥、法律責任時標註限制並建議轉介。

## 零、AI Agent 使用指引

> **讀取順序**：① 本節 → ② 使用者提供之個案資料 → ③ 依需要查閱下列各章 → ④ 輸出至 `個案概念化.md` 各欄位。

### 0.1 輸入與輸出

| 項目 | 說明 |
|------|------|
| **知識來源** | 本檔案（臨床概念、理論、衡鑑派別） |
| **格式模板** | [個案概念化.md](./個案概念化.md) |
| **輸出語言** | 繁體中文、專業但可讀 |
| **缺失資料** | 填「待評估」或留空，**不可捏造**症狀、史實、測驗分數 |

### 0.2 撰寫流程

```
1. 萃取主訴、現況、背景、風險、測驗（若有）
2. 填寫模板「一～四、六」之事實描述（行為具體化）
3. 依指示或個案特質選理論取向（見第十一章）
4. 撰寫「五、理論取向與個案概念化」：形成／誘發／維持／保護
5. 連結「八、治療計畫」目標與介入（對應概念化）
6. 檢查：風險、保密、去識別化、不越權診斷
```

### 0.3 模板章節 ↔ 知識庫對照

| 個案概念化.md | 參考本庫章節 |
|---------------|--------------|
| 一、基本資料與主訴 | 三（晤談）、零（不臆造） |
| 二、現況評估（含 2.2 情緒模式） | **六（情緒心理學）**、四（MSE）、七（症狀） |
| 三、背景與脈絡 | **七（發展／Erikson）**、六.3（依附）、十三（文化） |
| 四、生物—心理—社會 | 十二 |
| 五、理論取向與概念化 | **十一（衡鑑派別）**、九、十 |
| 六、風險評估 | 十三 |
| 七、診斷性思考 | 八（非醫師僅思考） |
| 八、治療計畫 | 十、六（情緒調節） |

### 0.4 撰寫原則

- **區分事實與推論**：事實來自資料；推論用「可能」「推測」「尚待確認」
- **情緒描述**：優先使用個案用語；可補充情緒粒度、調節策略（見第六章）
- **理論整合**：可複選派別，但須**前後一致**；主派別決定第五章小節結構
- **惡性循環**：至少一條具體「情境→想法→情緒→行為→後果」（模板 5.4）
- **禁止**：確診式 DSM 斷言（非醫師）、虛構會談次數、虛構測驗分數

### 0.5 理論取向選擇啟發式

| 若資料突出…… | 優先派別 |
|--------------|----------|
| 自動化思考、迴避、行為循環 | 認知行為 |
| 早期關係、重複模式、防衛、夢／移情 | 心理動力 |
| 自我價值、意義、真實自我 vs 理想自我、被理解需求 | 人本／存在 |
| 親密恐懼、拋棄、照顧者不一致 | 依附 |
| 創傷、解離、高度警覺 | 創傷知情 + BPS |
| 未指定 | 以 **BPS 整合** 撰寫 5.2，派別標「待與督導確認」 |

## 一、專業倫理與法律

### 1.1 核心倫理原則

| 原則 | 臨床意涵 |
|------|----------|
| **善行／不傷害** | 介入須有利於個案；評估風險後行動，避免過度承諾或延誤轉介 |
| **自主／知情同意** | 說明評估目的、程序、保密限制、資料使用；未成年人須法定代理人同意 |
| **誠實／真實** | 報告如實呈現結果與限制；不偽造、不誇大效度 |
| **保密** | 除法定例外外不得洩漏；團體督導須去識別化 |
| **公平／正義** | 避免歧視；衡鑑工具須注意文化與語言適切性 |
| **專業能力** | 僅施測受訓且具資格之工具；超出能力範圍須轉介或共評 |

### 1.2 保密例外（台灣常見情境，依機構與法規為準）

- 個案有**立即且嚴重**自傷、自殺或傷人風險
- **兒少保護**：懷疑兒童或少年遭受虐待、疏忽、性侵害
- **法院命令**或依法應報告之情形
- 個案**書面同意**揭露（如轉介、學校 IEP、保險）

> 晤談／衡鑑開始前應口頭與書面說明保密及其限制，並記錄個案理解與同意。

### 1.3 專業界線

- 避免雙重關係（親友、商業、性親密）
- 禮物、金錢、社群媒體互動須符合倫理與機構政策
- 衡鑑者與治療者**宜分離**（避免治療關係影響客觀評估）

### 1.4 紀錄與保存

- 紀錄應**及時、客觀、具體**（行為觀察優於價值判斷）
- 保存年限依衛福部、學會及機構規定（常見 7 年以上）
- 個案有權申請閱覽紀錄（依法定程序）

## 二、心理測驗學基礎

> 心理衡鑑教科書核心：測驗如何建立、分數如何解釋、結果是否可信。臨床施測前應理解這些概念，避免「只看分數高低」。

### 2.1 心理測驗的定義與目的

| 概念 | 說明 |
|------|------|
| **測量** | 將心理特質以規則化方式量化（智力、情緒、人格等） |
| **測驗** | 標準化程序 + 計分規則 + 常模 |
| **衡鑑** | 測驗 + 晤談 + 觀察 + 紀錄 + 整合判斷（**多方法**） |

**衡鑑目的**：描述（現況）、預測（預後／風險）、解釋（成因機制）、介入規劃（治療標的）。

### 2.2 信度

> 測量的一致性、穩定性；信度低則分數解釋無意義。

| 類型 | 意義 | 臨床注意 |
|------|------|----------|
| **再測信度** | 隔時間重測之一致性 | 情緒、症狀會變，不宜期待過高 |
| **複本信度** | 兩平行版本之一致性 | 較少在臨床單次使用 |
| **內部一致** | 題目測同一構念（如 α） | 人格量表常報告；單一構念篩檢量表適用 |
| **評分者信度** | 不同施測／評分者一致 | Rorschach、智力、神經心理重要 |
| **概化理論** | 分離誤差來源 | 研究與進階衡鑑 |

**測量標準誤（SEM）**：真實分數可能落在觀察分數 ± SEM 範圍內；報告宜呈現**信賴區間**，避免過度精確解讀 1–2 分差距。

### 2.3 效度

> 測驗是否測到宣稱的東西、是否用對地方。

| 類型 | 問題 | 例 |
|------|------|-----|
| **內容效度** | 題目是否涵蓋構念？ | 專家審題、題目–構念對應 |
| **效標關聯** | 與外在標準相關？ | 同時效度（與現況診斷）、預測效度（預測學業、復發） |
| **建構效度** | 是否符合理論網絡？ | 因子分析、區辨效度、收斂效度 |
| **表面效度** | 看起來是否相關？ | 非充分條件，但影響受測動機 |
| **生態效度** | 是否反映真實生活？ | 行為觀察、日常功能量表 |

**效度威脅（臨床常見）**：

- **社會期許**：填答取悅他人或否認問題
- **偽造／誇大**：司法、補助、逃避責任
- **防衛**：投射、否認（非故意作假但影響自陳）
- **文化／語言**：翻譯、常模不符、教育程度
- **身心狀態**：疲勞、疼痛、物質、急性精神病

### 2.4 常模與分數類型

| 分數 | 說明 | 臨床解讀 |
|------|------|----------|
| **原始分** | 答對題數、量表加總 | 須轉換常模才有意義 |
| **百分等級** | 同常模團體中的相對位置 | 易理解；極端值注意天花板／地板效應 |
| **標準分（z, T, IQ）** | 平均 50（T）或 100（IQ），標準差 10 或 15 | 比較**同一測驗內**分量表 |
| **年齡／年級當量** | 表現如幾歲 | 兒少常用；勿與實際年齡混淆 |
| **切截分數** | 篩檢陽性門檻 | 敏感度高但假陽性多，需後續確認 |

**常模選擇**：年齡、性別、教育、語言、文化背景應與常模族群相近；否則解釋須保守並註明限制。

### 2.5 剖面分析與整合

- **組型**：量表之間相對高低模式（如 MMPI 2-7-8）
- **水平**：整體偏高或偏低
- ****：變異大可能表不穩定或混合問題
- **剖面比較**：與常模、與個案過去測驗、與理論預期比較

> 教科書強調：**單一分量表不可單獨診斷**；須交叉驗證晤談、他評、觀察。

### 2.6 反應風格與效度指標

| 風格 | 表現 | 處理 |
|------|------|------|
| **偽好** | 否認問題、過度正向 | 效度量表（L、K）、晤談對質、他評 |
| **** | 誇大症狀 | F 量表、不一致題、情境合理性 |
| **隨機／不負責任** | VRIN、TRIN 高等（視工具） | 重測、建立動機、縮短工具 |
| **防衛性否認** | 非故意但低估 | 投射、行為觀察、信任關係後再測 |

**司法／職災／失能鑑定**：常需**效度／努力測驗**（如 PVT、SVT、TOMM、WMT）及結構式偽造評估，須受專門訓練。

### 2.7 行為評估

**ABC 分析**：

- **A**ntecedent：前因（情境、指令、內在想法）
- **B**ehavior：可觀察行為（頻率、強度、持續）
- **C**onsequence：後果（注意、逃避、獎賞）

**功能分析四類**：取得注意、逃避要求、取得 tangible、自我刺激（感覺）。

應用：ADHD、自閉症、對立反抗、職場問題行為；與認知測驗互補。

## 三、晤談與心理衡鑑流程

### 3.1 心理衡鑑的類型

| 類型 | 目的 | 常見情境 |
|------|------|----------|
| **診斷性衡鑑** | 釐清症狀、診斷、嚴重度 | 轉介精神科、司法、學校特殊需求 |
| **治療性／諮商性衡鑑** | 了解問題維持因素、治療標的 | 個案概念化、治療計畫 |
| **職業／教育衡鑑** | 能力、興趣、適性 | 升學、職涯、職能復健 |
| **神經心理衡鑑** | 認知功能、腦傷後影響 | 失智篩檢、TBI、注意力問題 |
| **司法／鑑定** | 責任能力、監護、再犯風險 | 法院委託（須特定訓練） |

### 3.2 標準衡鑑流程

```
接案與同意 → 初談／背景蒐集 → 選擇工具 → 施測 → 計分與解釋
    → 結果整合 → 口頭回饋 → 書面報告 → 追蹤／轉介
```

### 3.3 結構式初談要項

| 領域 | 蒐集內容 |
|------|----------|
| 主訴與期待 | 來談原因、目標、緊迫性 |
| 現況 | 情緒、認知、行為、睡眠、食慾、功能 |
| 精神病史 | 過去診斷、住院、自傷自殺、用藥 |
| 家族史 | 精神疾病、自殺、物質使用 |
| 發展史 | 懷孕、里程碑、學業、同儕、創傷 |
| 醫療史 | 慢性病、手術、用藥、物質 |
| 社會史 | 家庭、婚姻、工作、經濟、法律 |
| 文化與身份 | 語言、宗教、性別認同、移民經驗 |
| 優勢與支持 | 因應技巧、人際、興趣、價值 |

### 3.4 晤談技巧（諮商與衡鑑共通）

| 技巧 | 說明 | 範例 |
|------|------|------|
| **開放式提問** | 促進敘述 | 「可以多說一些那時候的感受嗎？」 |
| **反映** | 鏡映情緒或意義 | 「聽起來您覺得很孤單。」 |
| **澄清** | 釐清模糊處 | 「您說的『受不了』是指情緒還是身體？」 |
| **摘要** | 整合重點 | 「目前我聽到三件事……」 |
| **具體化** | 行為化、情境化 | 「最近一次發生是什麼時候？做了什麼？」 |
| **正常化**（謹慎使用） | 減少羞恥 | 「很多人在壓力下也會有類似反應。」 |
| **沉默** | 留空間處理情緒 | 不急於填補 |
| **對質**（建立關係後） | 指出不一致 | 「您說沒關係，但談到時眼眶紅了。」 |

**衡鑑晤談注意**：保持中立、結構清楚、控制時間；情緒激動時先穩定再施測。

---

### 3.5 轉介問題導向的衡鑑計畫（課本核心）

> **先問「要回答什麼問題」，再選工具**——避免制式化全套施測。

| 轉介問題類型 | 衡鑑焦點 | 可能工具／方法 |
|--------------|----------|----------------|
| 是否為 ADHD？ | 注意力、衝動、跨情境、起病史 | 晤談、Conners、CBCL、智力（排除學習問題） |
| 憂鬱嚴重度與自殺風險？ | 症狀、功能、風險、共病 | PHQ-9、BDI、結構式風險晤談、MSE |
| 學習困難原因？ | 智力、成就、注意力、情緒 | WISC + 成就測驗、晤談、學校紀錄 |
| 人格／適應問題？ | 長期模式、人際、防衛 | MMPI、MCMI、TAT、晤談 |
| 認知衰退？ | 記憶、執行、定向、日常功能 | MoCA、神經心理電池、病史、影像（醫師） |
| 職涯適性？ | 興趣、能力、性格、價值 | 興趣量表、智力、人格、晤談 |

**衡鑑計畫書寫要素**：評估目的、假設、方法、時程、預期產出（報告、會議）。

## 四、精神狀態檢查（MSE）

> Mental Status Examination：觀察當下精神狀態，為臨床與衡鑑必備。

| 項目 | 觀察重點 | 異常線索 |
|------|----------|----------|
| **外觀** | 服儀、衛生、年齡感 | 明顯邋遢、奇異打扮 |
| **行為／態度** | 合作度、眼神、動作 | 敵意、退縮、過動、怪異動作 |
| **言語** | 速率、音量、流暢度 | 貧乏、加速、迫促、口吃加重 |
| **情緒（Mood）** | 個案自述感受 | 憂鬱、焦慮、易怒、欣快 |
| **情感（Affect）** | 可觀察之情緒表達 | 平淡、不適切、易變 |
| **思維歷程** | 思考流暢度 | 遲緩、奔逸、阻塞 |
| **思維內容** | 主題、信念 | 妄想、固著、自殺意念 |
| **知覺** | 幻覺、解離 | 聽幻、視幻、非真實感 |
| **認知** | 定向、記憶、計算、抽象 | 定向障礙、近事記憶差 |
| **洞察／判斷** | 對問題理解、決策 | 病識感差、衝動決策 |

### 4.1 定向感（常測）

- **人**：姓名、年齡
- **地**：何處
- **時**：日期、季節
- **事**：為何而來

### 4.2 思維內容紅旗

- 自殺／自傷意念或計畫
- 他傷、命令式幻聽
- 被害、關係、誇大、宗教妄想
- 物質使用、譫妄徵象

## 五、心理衡鑑工具總覽

## 六、情緒心理學

> 個案概念化必備：情緒如何被**經驗、命名、調節、表達**，以及與認知、行為、關係之連結。填寫模板 **§2.2 情緒模式** 時優先參考本章。

### 6.1 情緒、情感、心情（Mood）之區分

| 用語 | 定義 | 衡鑑／概念化用法 |
|------|------|------------------|
| **情緒（Emotion）** | 短暫、具情境、含主觀＋生理＋行為傾向 | 「被批評當下」的羞恥、憤怒 |
| **情感（Affect）** | 可觀察的情緒**表達**（MSE） | 平淡、易哭、與 mood 不一致 |
| **心情（Mood）** | 較持久、彌散的情緒基調 | 憂鬱、易怒達數週 |

### 6.2 基本情緒理論

| 取向 | 重點 | 臨床意涵 |
|------|------|----------|
| **Ekman 基本情緒** | 快樂、悲傷、憤怒、恐懼、厭惡、驚訝（＋輕蔑） | 辨識情緒類型、跨文化表情 |
| **Plutchik 情緒輪** | 八.primary 情緒＋強度、對極、混合 | 理解情緒組合（如焦慮＝恐懼＋期待） |
| **Russell 環狀模型** | 效價（愉悅—不愉悅）× 喚起（激動—平靜） | 描述「又焦又累」等高喚起負向情緒 |
| **構建論（Barrett 等）** | 情緒由大腦**預測＋身體訊號＋情境概念化**共同建構 | 與情緒粒度、語言標記密切相關 |

### 6.3 情緒粒度

> **定義**：區分並精確命名情緒狀態的能力（用「羞愧、失落」而非籠統「不舒服」）。

| 粒度 | 表現 | 可能臨床關聯 |
|------|------|--------------|
| **高粒度** | 能區分相似情緒、描述誘因與身體感 | 較佳情緒調節、較少以酗酒等宣洩 |
| **低粒度** | 僅「糟透了」「很煩」 | 反芻、外化、軀體化、人際誤解 |

**衡鑑問句（Agent 可據資料改寫）**：

- 當時**具體**是什麼感受？（若只說「難過」，追問：傷心、空虛、憤怒、害怕？）
- 身體哪裡有感覺？強度 0–10？持續多久？
- 情緒出現前後，**想法**與**行為**各是什麼？

**概念化書寫範例**：「個案情緒粒度偏低，常以『壓力大』統稱焦慮與憤怒，導致難以選擇適切因應，傾向壓抑或爆發。」

### 6.4 情緒調節（Gross 歷程模型）

| 階段 | 策略 | 例 | 適應性 |
|------|------|-----|--------|
| **情境選擇** | 接近／迴避 | 避開社交 | 短期降焦慮、長期維持社交恐懼 |
| **情境修正** | 改變情境 | 與上司溝通 | 問題解決導向 |
| **注意部署** | 分心、專注 | 反芻 vs 正念 | 視內容而定 |
| **認知改變** | 重評、再詮釋 | CBT 認知重建 | 核心 CBT 機制 |
| **反應調節** | 壓抑、宣洩、放鬆 | 壓抑表情、運動 | 壓抑常短期有效、長期耗損 |

**概念化**：標示個案**慣用**哪幾階段、何者有效／無效，連結維持因素（如「情境選擇＝廣泛迴避」）。

### 6.5 情緒相關臨床現象

| 現象 | 說明 | 概念化提示 |
|------|------|------------|
| **述情障礙（Alexithymia）** | 難辨識、描述情緒；偏重軀體 | 常見於創傷、身心症、物質使用 |
| **情緒迴避** | 害怕感受而忙碌、物質、解離 | ACT「經驗迴避」、維持因素 |
| **情緒洪水（Flooding）** | 強度過高、難以調節 | DBT、創傷、邊緣特質；窗戶容忍度 |
| **窗戶容忍度（Window of Tolerance）** | 過低→解離／麻木；過高→失控 | 創傷知情概念化核心 |
| **情緒推理** | 「我感到害怕→一定有危險」 | CBT 認知扭曲 |
| **情緒感染／共調** | 人際情緒相互影響 | 家庭、伴侶系統 |

### 6.6 情緒與其他系統之整合

```
誘發情境 → 身體喚起 → 情緒標記（粒度）→ 意義／想法 → 調節策略 → 短期後果 → 長期模式
```

**Agent 填寫 §2.2 情緒模式 檢核**：

- [ ] 主要情緒（具體用詞，非籠統）
- [ ] 誘發情境（時間、地點、人物）
- [ ] 強度、持續、身體感
- [ ] 調節方式及後果（有效／無效）
- [ ] 與 §2.3 認知、§2.4 行為是否連貫

## 七、發展心理學要點

### 7.1 Erikson 心理社會危機（完整表）

| 階段 | 年齡 | 危機 | 成功結果 |
|------|------|------|----------|
| 信任 vs 不信任 | 0–1 | 照顧可靠性 | 希望 |
| 自主 vs 羞恥 | 1–3 | 自主探索 | 意志 |
| 主動 vs 內疚 | 3–6 | 主動遊戲 | 目的 |
| 勤奮 vs 自卑 | 6–12 | 學業、同儕 | 能力 |
| 認同 vs 角色混淆 | 12–18 | 我是誰 | 忠誠 |
| 親密 vs 孤立 | 成年早期 | 親密關係 | 愛 |
| 生產 vs 停滯 | 中年 | 工作、家庭貢獻 | 關懷 |
| 整合 vs 絕望 | 老年 | 生命回顧 | 智慧 |

### 7.2 Erikson 未解危機與成人表現

> 非「失敗」標籤，而是理解**當前困擾與發展任務**的框架；須有童年／青少年史料支持。

| 階段／危機 | 若早年不利經驗 | 成人常見表現 | 情緒／關係主題 |
|------------|----------------|--------------|----------------|
| 信任 vs 不信任 | 照顧不穩、虐待 | 難信他人、警覺、依附焦慮 | 恐懼、孤單 |
| 自主 vs 羞恥 | 過度批判、控制 | 怕犯錯、拖延、依賴 | 羞恥、無力 |
| 主動 vs 內疚 | 壓抑探索、懲罰 | 不敢主動、過度順從 | 內疚、退縮 |
| 勤奮 vs 自卑 | 學業羞辱、比較 | 成就焦慮、冒牌者感 | 自卑、嫉妒 |
| 認同 vs 混淆 | 角色期待衝突 | 方向感低、認同混亂 | 空虛、同儕壓力 |
| 親密 vs 孤立 | 被拒、背叛 | 親密恐懼、孤立 | 失落、迴避親密 |
| 生產 vs 停滯 | 職涯／育兒挫折 | 空轉、後悔 | 無意義、煩躁 |
| 整合 vs 絕望 | 未處理失落 | 絕望、未竟事務 | 悲傷、後悔 |

**Agent**：依個案**年齡**標示當前可能之發展任務（如 28 歲→親密；50 歲→生產），並連結主訴。

### 7.3 依附理論（Bowlby / Ainsworth）

| 類型 | 照顧者模式 | 成人關係傾向 |
|------|------------|--------------|
| **安全** | 敏感、一致 | 親密與自主平衡 |
| **焦慮—矛盾** | 不一致 | 害怕被拋棄、過度需求 |
| **逃避—拒絕** | 拒斥、冷淡 | 迴避親密、壓抑需求 |
| **混亂** | 恐懼／虐待 | 關係矛盾、創傷重現 |

> 臨床宜以「**內在運作模式**」描述，避免標籤化個案。

### 7.4 發展里程碑（篩檢用，個別差異大）

| 領域 | 需留意之遲緩 |
|------|--------------|
| 語言 | 2 歲少於 50 詞、無二詞句；不指物、不眼神接觸 |
| 動作 | 明顯坐、走、精細動作落後 |
| 社交 | 少共同注意、假扮遊戲、同儕互動 |
| 認知 | 學業持續落後、無法理解簡單指令 |

### 7.5 青少年與老年特別議題

- **青少年**：認同、同儕、網路、性、自傷、冒險行為；保密與家長告知平衡
- **老年**：失落、身體衰退、失智疑慮、孤獨、多重用藥、自殺（尤其老年男性）

## 八、常見精神病理與症狀群

### 8.1 情緒障礙

| 疾患 | 核心特徵 | 鑑別 |
|------|----------|------|
| **重度憂鬱** | 憂鬱情緒或快感缺失 ≥2 週，多項症狀、功能損害 | 雙相、哀傷、甲狀腺、物質 |
| **持續性憂鬱** | 慢性低落 ≥2 年 | 性格、環境壓力 |
| **雙相** | 躁／輕躁發作 + 憂鬱史 | ADHD、物質、精神分裂 |
| **焦慮症** | 過度恐懼／焦慮，迴避，功能損害 | 身體疾病、物質、憂鬱 |

### 8.2 焦慮相關（常見亞型）

- **廣泛性焦慮**：多面向、難控制、身體緊繃、易疲勞
- **恐慌症**：突發恐慌發作 + 持續擔心再發作
- **特定恐懼、社交焦慮**：情境觸發、迴避
- **強迫症**：侵入性思維 + 強迫行為／精神儀式
- **創傷後壓力**：暴露於創傷後，再體驗、迴避、警覺、負向認知情緒

### 8.3 精神病性障礙

- **妄想、幻覺、解離思維、言行紊亂、陰性症狀**
- 須排除：物質、醫學、情緒疾患伴精神病特徵
- 初發精神病：**及早轉介精神科**，考量神經發育與家族史

### 8.4 物質相關

- 評估：種類、劑量、頻率、戒斷、合併使用、復發史
- 常見：酒精、尼古丁、大麻、興奮劑、鴉片類
- 共病：憂鬱、焦慮、人格疾患、創傷

### 8.5 神經發展與神經認知

| 疾患 | 重點 |
|------|------|
| **ADHD** | 童年起病、多情境、注意／過動衝動 |
| **自閉症類** | 社交溝通缺陷、侷限重複行為（早發） |
| **學習障礙** | 特定學業成就顯著低於智力預期 |
| **失智** | 認知衰退影響功能，進行性 |

### 8.6 飲食與身體相關

- **厭食**：限制攝取、體重恐懼、體像扭曲
- **暴食**：反覆暴食 + 不當補償（催吐等）
- **身症症、疾病焦慮**：身體症狀或疾病恐懼主導

### 8.7 人格障礙（Cluster 概念）

| Cluster | 特質 | 範例 |
|---------|------|------|
| A | 古怪、退縮 | 妄想型、精神分裂型 |
| B | 戲劇化、不穩定 | 邊緣、反社會、自戀、戲劇 |
| C | 焦慮、僵化 | 回避、依賴、強迫 |

> 人格診斷需**長期模式**，青少年慎用；治療關係與界線尤重要。

## 九、診斷分類系統（DSM-5-TR／ICD-11）

### 9.1 使用原則

- **非醫師**在台灣通常不正式下 DSM／ICD 診斷；可做**診斷性思考**供轉介
- 診斷是**假設**，隨資料修正；重視功能損害與文化脈絡
- **共病常態**：憂鬱 + 焦慮、PTSD + 物質、ADHD + 對立反抗

### 9.2 DSM-5-TR 章節速查

| 章節 | 內容 |
|------|------|
| 神經發展 | 自閉、ADHD、學習、智能 |
| 精神分裂譜系 | 精神分裂、譜系、譫妄性 |
| 雙相與相關 | 雙相 I／II、環性 |
| 憂鬱 | 重度、持續、經前、物質／醫學 |
| 焦慮 | 特定、社交、恐慌、廣泛、物質 |
| 強迫及相關 | OCD、身型、囤積、拔毛 |
| 創傷與壓力 | PTSD、急性、適應障礙 |
| 解離 | 解離性身分、解離性健忘 |
| 身症 | 身症、疾病焦慮、轉換 |
| 餵食與飲食 | 厭食、暴食、暴食症 |
| 排泄 | 遺尿、遺便 |
| 睡眠—覺醒 | 失眠、嗜睡、呼吸相關 |
| 性功能 | 多種性功能障礙 |
| 性別煩悶 | 性別認同相關困擾 |
| 破壞、衝動控制 | 對立反抗、反社會、間歇性爆炸 |
| 物質 | 酒精、藥物、賭博 |
| 神經認知 | 譫妄、輕度認知障礙、失智 |
| 人格 | 10 種特定人格疾患 |
| 其他 | 自傷、文化症候群等 |

### 9.3 五軸精神（歷史）→ 現行重點

現行 DSM-5 強調：

1. **診斷**（含嚴重度、精神病特徵標註）
2. **文化因素**（Cultural Formulation Interview）
3. **殘疾與功能**（WHODAS 等）
4. **心理社會壓力**（V／Z codes）
5. **共病與治療建議**

### 9.4 鑑別診斷常用框架

```
Step 1：是否為物質／醫學所致？
Step 2：是否為精神病性疾患？
Step 3：是否為雙相（躁狂史）？
Step 4：是否為焦慮／強迫／創傷？
Step 5：是否為神經發展或認知？
Step 6：是否為人格或適應問題？
```

## 十、人格理論與評估

### 10.1 大五性格（OCEAN）

| 特質 | 高 | 低 |
|------|----|----|
| **開放性** | 好奇、創意 | 務實、傳統 |
| **盡責性** | 有序、可靠 | 隨性、易拖延 |
| **外向性** | 熱情、主動 | 內向、喜獨處 |
| **親和性** | 合作、信任 | 競爭、多疑 |
| **神經質** | 情緒易波動 | 情緒穩定 |

### 10.2 心理動力：防衛機制（常見）

| 機制 | 說明 |
|------|------|
| 壓抑 | 推開意識無法接受的衝動 |
| 否認 | 拒絕承認現實 |
| 投射 | 將己所不能接納的歸於他人 |
| 合理化 | 找理由掩飾真實動機 |
| 昇華 | 將衝動轉為社會可接受形式 |
| 反向形成 | 表現與內在相反 |
| 理智化 | 過度分析迴避情緒 |
| 分裂 | 全好／全壞看待自我與他人 |

### 10.3 認知：貝克認知層級

```
早期經驗 → 核心信念 → 中間信念（規則、假設）→ 自動化思考 → 情緒／行為
```

**常見認知扭曲**：全有全無、過度概括、讀心術、災難化、應該陳述、情緒推理、標籤化、心理過濾、貶低正向。

## 十一、心理衡鑑理論派別與概念化

> **本章為 Agent 撰寫「個案概念化.md §五」之核心**。心理衡鑑與概念化可依不同**理論透鏡**組織同一批資料；派別影響：問什麼、看什麼、如何解釋維持因素、如何寫治療目標。

### 11.1 三大派別總覽

| 派別 | 核心假設 | 衡鑑焦點 | 常用工具／方法 |
|------|----------|----------|----------------|
| **心理動力** | 無意識衝突、早期關係模板重現 | 發展史、防衛、移情、內在客體 | 結構式晤談、TAT、Rorschach、MMPI 內容 |
| **認知行為（CBT）** | 認知—情緒—行為互為因果、維持循環 | 自動化思考、信念、迴避、強化 | 功能性分析、自陳量表、行為觀察、SUDS |
| **人本／存在** | 自我實現、主觀經驗、意義與真實自我 | 現象學描述、價值、阻礙、關係品質 | 深度晤談、此時此地、個案語言 |

其他常整合：**依附**、**系統／家庭**、**創傷知情**、**BPS**（見第十二、十四、十五章）。

### 11.2 心理動力取向（概念化與衡鑑）

**評估要項**：

| 領域 | 蒐集內容 | 寫入模板 |
|------|----------|----------|
| 早期關係 | 照顧者、依附、創傷、同胞 | §3.1 |
| 重複模式 | 人際、職場、權威關係之重演 | §5.3 心理動力小節 |
| 防衛機制 | 壓抑、投射、分裂等（見第十章） | §5.3 |
| 內在衝突 | 欲望 vs 超我、依賴 vs 自主 | §5.2 形成因素 |
| 移情／反移情 | 對治療者、重要他人之反應 | §5.3、§8.4 督導 |

**概念化敘述架構**：

> 早年…… → 形成內在客體／自我意象 → 成年後在……情境重現 → 以……防衛處理情緒 → 短期……但長期維持……

**維持因素常見**：迴避哀傷、投射敵意、關係中重現父母角色、症狀的次要獲益（需謹慎推論）。

### 11.3 認知行為取向（概念化與衡鑑）

**評估要項**：

| 領域 | 蒐集內容 | 寫入模板 |
|------|----------|----------|
| 觸發情境 | 何人、何地、何時 | §2.2、§5.4 |
| 自動化思考 | 熱點想法、信念 | §2.3 表格 |
| 情緒／生理 | 種類、強度、身體 | §2.2、§6 章 |
| 行為 | 迴避、安全行為、儀式 | §2.4 |
| 後果 | 短期降焦慮、長期功能下降 | §5.4 |
| 核心／中間信念 | 關於自我、他人、世界 | §2.3、§5.3 CBT |

**惡性循環（必填一條，模板 5.4）**：

```
情境 → 自動化思考 → 情緒／生理 → 行為（常為迴避或安全行為）→ 短期後果 → 長期維持（信念被強化）
```

**衡鑑工具**：BDI、BAI、功能分析、活動紀錄、信念問卷（若有資料才寫分數）。

### 11.4 人本／存在取向（概念化與衡鑑）

> 模板未單列小節時，寫入 **§5.2 概念化敘述**，並在 **§5.1** 勾選「人本／存在」。

**評估要項**：

| 領域 | 蒐集內容 | 說明 |
|------|----------|------|
| **主觀經驗** | 個案如何描述活在世界上的感受 | 現象學：不急于解釋，先描述 |
| **自我概念** | 理想自我 vs 真實自我、不一致 | Rogers 不一致（incongruence） |
| **阻礙** | 條件式價值、他人評價內化 | 「我應該……才能被愛」 |
| **意義** | 價值、目的、失落 | 存在：空虛、死亡、自由、孤獨 |
| **關係** | 被理解、被接納、真實表達 | 治療關係為改變媒介 |
| **優勢** | 自我決定、過去因應、資源 | §4.4 保護因子 |

**概念化敘述架構**：

> 個案在……處境中感到……（主觀體驗）。為符合……期待，壓抑……真實感受，導致……（症狀／困擾）。其深層需求為……（被看見、意義、自主）。當……支持存在時，較能……

**與 CBT 差異**：少標籤「扭曲」，多描述**意義與價值衝突**；與心理動力差異：少強調無意識衝突詮釋，多強調**此時此地經驗與選擇**。

### 11.5 依附取向（補充，常與上述整合）

- **內在運作模式**：自我模型（值得／不值得被愛）× 他人模型（可信／不可信）
- **情緒調節**：安全基地、抗壓、探索；不安全時之抗爭或迴避
- **寫入**：模板 §5.3 依附小節；連結第六章情緒、第七章 Erikson／依附

### 11.6 整合性概念化（推薦預設）

當資料複雜或未指定派別，採 **BPS + 主派別**：

1. **生物**：睡眠、用藥、家族史、慢性病
2. **心理**：依主派別寫認知／情緒／動力（第六章＋第十章＋11.2–11.4）
3. **社會**：壓力、文化、經濟、支持
4. **形成／誘發／維持／保護** 四段（模板 §5.2）

- [ ] §5.1 已勾選派別（與內容一致）
- [ ] §5.2 含形成、誘發、維持、保護（各至少一點，有資料才寫）

## 十二、諮商與心理治療取向

### 12.1 取向對照

| 取向 | 核心假設 | 常用技術 | 適用議題 |
|------|----------|----------|----------|
| **認知行為（CBT）** | 想法影響情緒行為 | 認知重建、行為實驗、曝露、行為活化 | 憂鬱、焦慮、OCD、失眠 |
| **接納承諾（ACT）** | 心理彈性、價值行動 | 正念、脫融合、價值澄清 | 慢性痛、焦慮、迴避 |
| **辯證行為（DBT）** | 情緒調節、接納與改變辯證 | 正念、痛苦耐受、人際效能 | 邊緣、自傷、情緒失調 |
| **心理動力** | 無意識衝突、早期關係 | 澄清、詮釋、移情工作 | 人際重複、性格 |
| **人本／存在** | 自我實現、意義 | 同理、此時此地、意義探索 | 存在焦慮、自我價值 |
| **依附取向** | 內在運作模式 | 情緒調節、安全基地 | 親密、創傷 |
| **EFT（情緒聚焦）** | 情緒為改變關鍵 | 空椅、標記情緒 | 伴侶、依附創傷 |
| **系統／家族** | 互動維持問題 | 循環質問、作業、重框 | 家庭衝突、青少年 |
| **創傷知情（TF-CBT, EMDR 等）** | 創傷記憶再處理 | 穩定化、曝露、EMDR | PTSD、複雜創傷 |

### 12.2 常見介入技術速查

| 技術 | 做法摘要 |
|------|----------|
| **認知重建** | 辨識自動化思考 → 證據檢驗 → 替代想法 |
| **行為活化** | 安排愉悅／掌控活動，打破退縮循環 |
| **系統化減敏／曝露** | 階梯式面對恐懼，阻斷迴避 |
| **正念** | 不評判覺察當下，與想法脫鉤 |
| **問題解決** | 定義問題 → 列方案 → 評估 → 執行 |
| **壓力接納與痛苦耐受** | 危機生存技巧、分心、自我安撫 |
| **空椅** | 角色對話處理未完成情緒 |
| **Homework** | 會間練習以泛化學習 |

### 12.3 治療階段（一般）

1. **建立關係與評估**
2. **目標設定與心理教育**
3. **核心介入**
4. **鞏固與復發預防**
5. **結案與追蹤**

### 12.4 治療聯盟（Wampold 等研究）

- **同理、真誠、正向關懷**為跨學派共同因子
- 早期聯盟品質預測療效；破裂時宜直接修復（rupture-repair）

## 十三、危機評估與介入

### 13.1 自殺風險評估（至少涵蓋）

| 面向 | 問題方向 |
|------|----------|
| **意念** | 頻率、強度、可控制性 |
| **計畫** | 具體方法、取得手段 |
| **意圖** | 是否打算執行 |
| **既往** | 過去自傷自殺、住院 |
| **誘發** | 近期失落、羞辱、物質 |
| **保護因子** | 子女、宗教、治療承諾、支持 |
| **精神狀態** | 精神病、衝動、物質中毒 |

### 13.2 風險分級與行動（原則）

| 等級 | 特徵 | 行動 |
|------|------|------|
| **低** | 被動意念、計畫模糊、保護因子足 | 安全計畫、密切追蹤、簽約 |
| **中** | 主動意念、部分計畫、支持不足 | 增頻會談、家屬參與、精神科 |
| **高** | 明確計畫意圖、手段可得、嚴重症狀 | **不單獨離開**、轉急診／住院、移除手段 |

### 13.3 安全計畫要素（Stanley & Brown）

1. 辨識警告徵象
2. 內在因應（正念、分散）
3. 社交聯絡人與電話
4. 專業資源（諮商師、急診、1925、1995、110）
5. 環境安全（移除藥物、武器）

### 13.4 其他危機

- **家暴／兒虐**：法定通報、安全優先、資源連結
- **急症精神病**：轉精神科急診、避免對質妄想
- **物質戒斷**：酒精、苯二氮卓戒斷可致命，須醫療

## 十六、衡鑑報告撰寫與整合

### 16.1 多元資料整合

**整合性衡鑑**指將不同來源資料收斂為一致（或說明不一致）的臨床判斷：

| 資料來源 | 提供資訊 | 整合時自問 |
|----------|----------|------------|
| 晤談 | 主觀經驗、史實、關係 | 與測驗是否一致？不一致原因？ |
| 自陳量表 | 自覺症狀、人格 | 效度指標是否可接受？ |
| 他評 | 外在功能、行為 | 與自評差異是否具意義？ |
| 認知測驗 | 能力、優弱項 | 成就落差是否支持學習障礙？ |
| 紀錄 | 學校、醫療、司法 | 時間序是否支持現行假設？ |
| 觀察 | 合作度、努力、MSE | 是否解釋測驗表現？ |

**不一致時的處理**：

1. 檢查施測條件（疲勞、動機、語言、效度）
2. 回到行為具體例證（何時、何地、多嚴重）
3. 在報告中**如實呈現**矛盾，提出 2–3 個可能解釋
4. 避免強行選一個分數「定案」

### 16.2 報告結構

1. **識別資料**（去識別化代號）
2. **轉介問題與評估目的**
3. **評估方法**（工具、日期、施測情境）
4. **相關背景**（精簡）
5. **行為觀察**
6. **測驗結果**（分數 + 解釋，非僅表格）
7. **整合與印象**
8. **建議**（治療、教育、職能、轉介）
9. **限制**（動機、效度、語言、單次施測）

### 16.3 撰寫原則

- **行為描述**優於標籤（「在算術題放棄」優於「懒惰」）
- 區分**事實、推論、建議**
- 避免**絕對化**（「一定」「永遠」）
- 提供**可執行**建議（誰、做什麼、多久）

### 16.4 口頭回饋

- 先**優勢與支持**，再談困難
- 避免專業術語轟炸；確認理解
- 敏感結果（人格、智力）宜私密、漸進
